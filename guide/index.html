全部运行

npm run dev

npm run build

访问和打包方式:

localhost:8080/projectName/index.html


运行单个项目

npm run dev --projectName(npm run dev --demo)

npm run build --projectName(npm run dev --demo)

访问和打包方式：

localhost:8080/index.html


项目结构

src/project/新项目文件夹

必备文件为：1.index.js 2.index.html

入口为index.js

每个页面文件夹必须包含 README.md 说明文件夹使用途径
